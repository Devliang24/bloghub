# 📚 Devliang24 BlogHub

> **高质量中文全栈技术教程库**  
> 专注于现代 Python 开发、容器化技术与云原生架构。

## 🌟 项目简介

本项目汇集了一系列精心编写的技术教程，旨在帮助开发者快速掌握现代 Web 开发的核心技能。所有文章均采用 **Markdown** 格式编写，并包含丰富的 **Mermaid 流程图**，力求将复杂的概念可视化、简单化。

特别针对**国内网络环境**进行了优化，所有实战示例均包含国内镜像源配置指南。

## 📂 内容导航

### 🚀 [FastAPI 完整实战教程](fastapi/) (20 篇)

从零开始掌握 Python 下一代高性能 Web 框架。

*   **入门基础**：环境搭建 (UV)、Hello World、参数校验、Pydantic 模型。
*   **核心进阶**：依赖注入系统 (Dependency Injection)、OAuth2 安全认证 (JWT)。
*   **数据库集成**：使用 **SQLModel** 进行 ORM 映射与 CRUD 操作。
*   **高级特性**：异步后台任务、中间件、CORS、WebSocket 实时通信。
*   **工程化**：自动化测试 (Pytest)、Docker 部署、大型项目目录结构规范。

### 🐳 [Docker 快速入门教程](docker/) (6 篇)

容器化技术的最佳实践指南。

*   **基础概念**：架构图解、镜像与容器生命周期管理。
*   **环境配置**：Windows/Linux 安装及**国内镜像加速器**配置。
*   **构建优化**：编写高效 Dockerfile、多阶段构建 (Multi-stage builds)。
*   **编排实战**：使用 Docker Compose 管理多容器应用 (FastAPI + MySQL + Redis)。
*   **存储与网络**：数据持久化 (Volume) 与容器互联。

## ✨ 教程特色

*   🇨🇳 **全中文编写**：通俗易懂，贴合国内开发者习惯。
*   📊 **图文并茂**：关键概念配有 Mermaid 流程图、时序图、状态机图。
*   ⚡ **实战导向**：拒绝纸上谈兵，提供可直接运行的代码示例。
*   🛠️ **工具现代化**：使用 **UV** 替代 Pip，使用 **SQLModel** 替代 SQLAlchemy 原生语法。

## 📖 如何使用

你可以直接在 GitHub 上浏览，或者将仓库克隆到本地：

```bash
git clone https://github.com/Devliang24/bloghub.git
```

所有文档均为标准的 Markdown 文件，推荐使用 VS Code 或 Typora 阅读以获得最佳体验。
